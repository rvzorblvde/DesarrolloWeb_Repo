HTML

El Lenguaje de Marcado de HiperTexto (HTML) es el código usado para estructurar y desplegar una página web y sus contenidos.

Al ser un lenguaje front-end, todo el código es accesible desde el navegador donde se esté ejecutando.

La estructura se basa en etiquetas que representan elementos, estos a su vez pueden tener atributos con sus respectivos valores



Estructura

Se componen de una etiqueta de apertura: <> y una etiqueta de cierre: </>

Ejemplo: <h1> Este es un encabezado </h1>


Algunas etiquetas pueden no incluir una etiqueta de cierre

También pueden incluir atributos con sus respectivos valores:

Ejemplo: <button type="submit">Aceptar</button>


Donde:

  • type es el atributo
  • "submit" es el valor para dicho atributo


Encabezados (h1-h6)

Los elementos de encabezado <h1></h1> implementan seis niveles de encabezado de documento, <h1> es el más importante y <h6> el menos importante.
