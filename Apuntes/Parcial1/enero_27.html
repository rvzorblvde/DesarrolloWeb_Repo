Tablas

Se pueden usar tablas para distribuir la información utilizando la etiqueta <table>.

  • Dentro de esta etiqueta usaremos <tr> (table row) para dividir cada renglón.
  • Dentro de cada renglón, se usa <td> (table data) para definir cada celda de la tabla.
  • <th> (table header) servirá para resaltar encabezados en nuestra tabla, se utiliza como el primer elemento de cada columna o renglón.
  • Por defecto todas las celdas ocupan el mismo espacio pero podemos establecer que ciertas celdas ocupen más espcacio a lo largo o ancho utlizando los atributos colspan y rowspan

Formularios

Uno de los principales puntos de interacción entre usuario y sitio web/aplicación. Permiten la introducción de datos que se envían a un servidor web para su procesamiento y almacenamiento.

Está compuesto por uno o más controles de formulario además de algunos elementos adicionales.

Los controles pueden ser campos de texto de varias líneas, cajas desplegables, botones, casillas de verificación o botones de opción y se crean principalmente con el elemento <input>.

principales elemenos para un formulario

  • <form>: Etiqueta principal que contiene todo el formulario y sus elementos, sus atributos principales son method, con el que definimos el método HTTP con el que viajarán los datos del formulario como post o get, y action que indica la ruta a donde se enviarán los datos.
  • <action>: Representa una etiqueta en nuestra interfaz y se suele ligar a un control input utilizando el atributo for y colocando como valor el ID de dicho elemento.

Elementos de control

El elemento HTML <input> se usa para crear controles interacitvos para formularios web con el fin de recibir datos del usuario.

Es uno de los elementos más potentes y complejos de HTML debido a su gran combinación de atirbutos y atributos de entrada

El atributo más importante de este elemento es type el cual podrá contener valores como text, password, number, date, radio, range, etc. dicho valor indicará que tipo de control será el elemento y su comportamiento.

Con el atributo name, le asignamos un nombre al elemento y por lo tanto a su valor, que posteriormente será con el que la información viajará cuando se envie.

Botones

La etiqueta <button> representa un elemento clickeable de tipo botón que puede ser utilizado en formularios o en cualquier parte de la página que necesite un botón estándar.
El atributo type nos indica el tipo de botón:

  • submit: Valor por defecto y con este indicamos que el botón va a interactuar directamente con el formulario que lo contiene y servirá para mandar la información capturada a la ruta definida en dicho formulario.
  • reset: Reestablece los valores iniciales de los controles en el formulario donde está incluido.
  • button:Desactivamos la interacción con el form (Si existe) y tendremos que agregar el comportamiento mediante eventos.
