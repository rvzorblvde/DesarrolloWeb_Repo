Enlaces o anclas

Crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página.

Se usa el atributo <href> para indicar la página o elmemento a enlazar.

Casos

  1. Si el enlace es una página, el valor de dicho atributo será la URL del sitio.

    Ejemplo: href="https://google.com/"

  2. Si el enlace es hacia un elemento, el valor será un signo de gato/hash (#) seguido por el ID de dicho elemento.

    Ejemplo: href="Inicio"

    Si aún no sabemos a donde redigirá el ancla, solamente ponemos un #.

Imágenes

El elemento <img> nos permite mostrar una imagen dentro de nuestra página.

  • Con el atributo src indicaremos la ubicación de la imagen que queremos mostrar (local o externa)
  • El atributo alt define un texto alternativo para describir la imagen en caso de que no se pueda mostrar.

<img src="https://picsum.photos/200/300" alt="Imagen de prueba"/>

Ejemplo 1: Imagen local

Se usa la ruta "./auto.jpg" para especificar la ruta del archivo local.

Ejemplo 2: Imagen de internet:

Se usa una dirección de internetpara obtener la imagen.

Ejemplo 3: Imagen no encontrada

Por cualquier motivo que no se pueda mostrar la imagen, se usa el texto definido en el atributo alt.

Listas

Se pueden organizar elementos utilizando <ol> para listas ordenadas y <ul> para listas no ordenadas.

La etiqueta <li> sirve para definir cada elemento de la lista (Puede ser texto u otros elementos).

Lista ordenada:

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Lista ordenada:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Puede incluir subelementos y combinar los tipos de listas.

Lista ordenada

  • Elemento 1
  • Elemento 2
    1. Subelemento 1
    2. Subelemento 2
    3. Subelemento 3
  • Elemento 3

Más etiquetas de texto

La etiqueta <b> nos sirve para resaltar texto en negrita.

<br> Nos permite dar saltos de línea y no requiere etiqueta de cierre.

La etiqueta <i> nos permie poner el texto en cursiva.
