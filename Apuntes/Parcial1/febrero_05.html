Propiedad Display

  • Especifica el trato que se le dará a un elemento en nuestra página para definir si compartirá o no espacio horizontal con otros elementos.
  • Los principales valores para esta Propiedad son block e inline, incluso los elementos por defecto ya cuentan con uno de estos dos valores asignados.

display: block

  • No pueden compartir espacio horizontal con otros elementos, por lo que los siguientes serán desplazados hacia abajo.
  • Admiten dimensiones (width, height) como margin, padder y border.
  • Suelen ser usados para generar estructuras, ya que pueden contener elementos hijos que sean inline.

display: inline

  • Permite que varios elementos compartan espacio horizontalmente.
  • Solo admiten margin, padding y border.
  • Si un elemento llega al límite horizontal se empuja hacia abajo.

Notas adicionales:

  • Se pueden combinar algunas características de ambos modos usando display: inline-block, de esta manera se puede tener un elemento de línea que si acepta valores de dimensiones.
  • También permite evitar que un elemento se muestre con display:none.
  • Existen valores más complejos que nos sirven para definir el comportamiento de los elementos hijos como display: flex.

Unidades de medida:

Podemos asignar a varias propiedades CSS un valor numérico con su respectiva unidad de medición.
Estas unidades tienen diferente efecto según como se estén utilizando.

  1. Pixeles (px): Unidad más básica con la que trabaja CSS para pintar elementos en el viewport y no siempre son del mismo tamaño que un pixel físico.
  2. Porcentaje (%): Establece el tamaño del elemento con respecto al tamaño del elemento padre.
